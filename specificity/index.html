CSS specificity

CSS specificity คือ การเลือกทำสิ่งที่มีลำดับความสำคัญมากกว่าเมื่อมี selector ที่ซ้ำกัน โดยจะเลือกทำอันที่มีความสำคัญมากที่สุด ยกตัวอย่างเช่น
